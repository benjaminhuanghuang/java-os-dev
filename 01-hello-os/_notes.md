Java代码构造一个1.44M (=147560Bytes) 的二进制文件, 

文件的前512个字节是FAT12文件系统信息和编译后的二进制可执行代码，这512个字节以 55AA 结束，表明这是一个启动扇区

BIOS可以加载启动扇区到内存的0x7c00并执行

```
 |---------------------------------------------------------------------------
 |                              0x168000（=1474560）                         |
 |---------------------------------------------------|-----------------------
 |          512 (0x000 ~ 0x1FF)byts                  |        Empty         |
 | --------------------------------------------------|                      |
 |     FAT32 文件系统 |  boot                    55 AA| F0 FF FF 00 00 ..... |
 |---------------------------------------------------|----------------------|   
```

此时如果在windows虚拟机中加载这个image到floppy disk driver，会看不到任何内容，做说明
1. 这个image中有windows可以识别的文件系统信息
2. 这个image中没有任何文件
